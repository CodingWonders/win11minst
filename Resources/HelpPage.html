Bugs:
- When using REGTWEAK as the installer creation method, and testing the custom installer on VMware, Setup will still block your system. In this case, please use one of the other methods. This is due to Setup still detecting components (if you're using builds 22000.194 or higher; or Nickel builds from 22458 onwards) on a hypervisor. This was tested on the leaked build (21996.1), and in VirtualBox; and this doesn't happen.

Instructions

1. Download the images (Windows 11 and Windows 10 images)

2. Click 'Create a modified WIndows 11 installer'

3. Specify the two Windows images

4. Specify where you want to save the custom image

5. Let the program do the magic

That's it!
